# Sound FX (bring your own)

This folder is **not** distributed with the repo — sound-effect sites usually
license their files for use *in your videos* but prohibit redistributing the
files themselves in a public collection, so FableCut doesn't ship any.

Drop `.mp3` / `.wav` files in here and they appear in the editor's **Sound FX**
tab instantly (the open UI live-refreshes). Subfolders are fine.

Good sources of free sounds:

- **freesound.org** — filter by the CC0 license for no-strings sounds
- **pixabay.com/sound-effects** — free for use in your projects
- **mixkit.co/free-sound-effects** — free for use in your projects

Handy starter set: a whoosh, an impact/boom, a pop, a camera shutter, a
typewriter tick, a riser, and a couple of UI clicks — most editing recipes in
CLAUDE.md reference sounds like these by role.
